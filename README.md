Overview

Decentralized Social Media (DeSoc) is a comprehensive research and product repository exploring how social networks can be rebuilt using blockchain, cryptography, and open protocols.

The project investigates how to move beyond Web2 platformsâ€”where user data, content, and algorithms are controlled by centralized corporationsâ€”toward user-owned, censorship-resistant, and interoperable social media systems.

Core Principles

User-owned identity (DID, wallets instead of accounts)

Content ownership (portable, censorship-resistant data)

Open social graph (no platform lock-in)

Transparent algorithms (or DAO-governed)

Creator-first monetization

Protocol over platform
